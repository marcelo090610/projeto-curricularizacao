# Projeto de Curricularização

Sistema de gestão desenvolvido como parte do projeto de extensão/curricularização. O objetivo é gerenciar alunos, salas e turmas.

## Equipe

* **Marcelo:** Backend e Estrutura (Banco de Dados, Migrations, Rotas)
* **Lucas:** Backend (Lógica de controle de presença, turmas e alunos)
* **Vinícius:** Frontend (Telas de autenticação, alunos e salas)
* **Higor:** Documentação e Diagramas

## Funcionalidades Principais

* **Autenticação:** Login e recuperação de senha.
* **Gestão de Alunos:** Cadastro, listagem e edição.
* **Gestão de Salas:** Controle de ocupação e salas disponíveis.
* **Controle Acadêmico:** Chamadas, presença e turmas.

## Tecnologias

* **Frontend:** (React)
* **Backend:** (PHP com Laravel Framework)
* **Banco de Dados:** (MySQL)
